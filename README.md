📊 连续劳务个税计算器（2025个税新规版）
一个专业、直观的在线工具，用于计算连续劳务报酬个人所得税，基于2025年最新个税政策预测。

🌟 项目特色
精准计算：依据《个人所得税法》及相关规定，采用累计预扣预缴算法

操作简便：可视化月份选择，支持批量或按月独立设置收入

结果清晰：详细展示每月税前、应预扣、税后收入及税负率

响应式设计：完美适配电脑、平板和手机等各种设备屏幕

权威信息：内置计税依据说明，方便用户查阅相关政策

🚀 快速开始
访问网站：[https://fengxiangtax.com](https://www.fengxiangld.com/)

或直接使用以下步骤：

在"灵活就业者收入"中输入月收入金额

选择需要计算的月份（可单选、多选或全选）

点击"开始计算"按钮

查看详细的月度计算结果和年度汇总

💡 主要功能
1. 灵活计算模式
默认模式：所有选中月份使用统一收入计算

高级模式：为每个月份单独设置不同收入金额

2. 智能计算逻辑
累计预扣预缴：按照连续劳务报酬计算规则

自动费用扣除：减除20%费用 + 每月5000元基本减除

累进税率：采用7级超额累进税率表

3. 完整结果展示
月度明细：每月税前、税款、税后、税负率

年度汇总：总收入、总税额、总税后、平均税负
